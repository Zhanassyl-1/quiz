# Binom School Quiz Displays

Landing experience for Binom School’s science week with two static HTML apps and CDN-only dependencies.

## Files
- `display.html`: fullscreen duo-view slideshow (auto-discovered `photos/` assets) plus Supabase-synced leaderboard and QR block linking to the quiz.
- `quiz.html`: mobile-first, single-file quiz that walks students through registration, Supabase question pools, bonus questions every third attempt, and leaderboard updates after results.
- `photos/`: repository of slideshow images consumed by `display.html`.

## Running Locally
Open `display.html` or `quiz.html` directly in a browser — no build step required, all dependencies load via CDN.
